# Simple Java Tasks

This repository contains beginner Java practice tasks.

## Task 1
Print all odd numbers up to 300.

## Task 2
Create an array `{1, 6, 7, 9, 5, 7, 3}` representing the number of toys in a store and an array `{2.5, 3.6, 8.9, 7.5, 2.0, 2.3, 6.3}` representing the price of one toy.  
Print the total value of all toys purchased.

## Task 3
Print the value of vacation expenses for a total budget of `10,000`, where:
- Accommodation is 50% of the total budget
- Beach equipment rental is 5% of the total budget
- Restaurant expenses are 30% of the total budget
- Additional entertainment is 10% of the total budget
- Other expenses are 5% of the total budget

## Task 4
Print all possible combinations of 5-character strings where:
- the first 2 characters are digits
- the next 2 characters are letters
- the fifth character is a digit

## Task 5
Write a program that prints all prime numbers in the interval from 1 to 300.

## Task 6
Write a program that prints what percentage of the numbers in the interval from 1 to 300 are divisible by the prime numbers in the same interval.
